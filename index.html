🔧 Jira Ikke tilkoblet

Henter saker tildelt deg.

⚠ CORS-begrensning: Jira blokkerer direkte API-kall fra nettlesere. Denne appen bruker CORS-proxyer som kan være ustabile. Hvis tilkoblingen feiler gjentatte ganger, vurder å bruke appen fra en server eller Jira CLI.

La feltet stå tomt for å beholde eksisterende token.

🔒 Legitimasjon lagres kun i nettleseren din.

📧 Microsoft 365 Ikke tilkoblet

Henter To-Do-oppgaver og kalenderhændelser.

Slik får du et token:
1. Gå til Microsoft Graph Explorer ↗
2. Klikk Sign in med din arbeidskonto
3. Klikk fanen Access token og klikk Copy
4. Lim inn i feltet nedenfor
Token utløper etter ~1 time. Bruk "Forny token" når det er nødvendig.

🔒 Token brukes kun i nettleseren din.

🚀 Del med teamet ditt — Publiser på GitHub Pages (gratis) ▼ vis

Ingen installasjon nødvendig — bare en gratis GitHub-konto.

1
Gå til github.com og logg inn (eller opprett gratis konto).
2
Klikk + → New repository. Gi det et navn, sett til Public, klikk Create.
3
Klikk uploading an existing file. Last opp denne filen som index.html. Commit.
4
Gå til Settings → Pages, velg branch main, klikk Save.
